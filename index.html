Meu gerenciador

Aplicação criada para treinar os conhecimentos adquiridos com JavaScript e aprender mais sobre Banco de Dados.

O objetivo é criar uma plataforma que seja capaz de acompanhar/gerenciar toda uma estrutura/cadeia produtiva. Serão implementadas soluções para gerenciar o estoque e lista de pedidos. Além de monitorar a produção em tempo real (ou algo próximo disso). Sendo então, uma plataforma que consegue fazer o gerenciamento (ou pelo menos o acompanhamento) completo da parte produtiva de uma empresa.

Optei por utilizar JavaScript ao invés de algum framework, para ver quais serão os problemas que aparecerão, e então analisar por conta própria a eficiência dos frameworks.

Primeiro será estruturado o Frontend, pois é a parte que estudei mais até então. Posteriormente, será criado o Backend juntamente com os ajustes que forem necessários no frontend.
